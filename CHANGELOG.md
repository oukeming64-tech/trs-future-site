# Changelog

## 1.2.0 — 2026-07-14

- 完成中科闻歌官网与 v1.1 的结构、内容、视觉和转化路径对比，并记录融合筛选与旧版/新版对跑结果。
- 为 24 个产品增加业务结果表达，为四类产品分别增加三步能力闭环。
- 为 7 个行业增加共 21 个典型业务场景，同时保留各自的独立 3D 动效。
- 在公司实力模块增加产品文档、生态体系与法定披露 3 个官方证据入口。
- 将新闻洞察更新为拓尔思官网 2026 年 7 月公开内容。
- 保持 7 个页面主模块、35 套 3D 场景和现有依赖数量不变，并补充内容完整性测试。

## 1.1.0 — 2026-07-14

- 将拓尔思品牌标识替换为官网原版 Logo 资源。
- 为全域感知、治理、认知与行动 4 个环节新增独立 3D 模型。
- 为 24 个产品分别新增与能力原型对应的微型 3D 动画语言。
- 为数字政府、金融科技、媒体融合、公共安全、企业数智化、知识产权与出版 7 个行业新增独立 3D 动效。
- 新增数据驱动的微型 3D 蓝图引擎、统一原语库、响应式规则与减少动态效果支持。
- 新增 35 套内容映射与场景蓝图的一致性测试，并同步架构与内容来源文档。
- 发布 55.83 秒中文解说首版视频，包含 7 段模块化叙事、烧录字幕、原创电子配乐与转场审校。

## 1.0.0 — 2026-07-14

- 建立拓尔思概念官网的完整信息架构。
- 新增实时 WebGL 3D 语义核心与鼠标、滚动联动。
- 新增大数据、人工智能、数据安全、云与应用产品切换。
- 新增七大行业方案、四类客户实践、公司实力、里程碑与新闻洞察。

- 移除 starter 的加载骨架、数据库、认证与示例代码。
- 更新 Cloudflare/Vite 构建链并锁定修复版 PostCSS，依赖审计为 0 项已知漏洞。
- 更新构建测试、项目文档、内容来源与开源许可。
- 发布站点所有者可访问的在线预览，并创建公开 GitHub 仓库。
